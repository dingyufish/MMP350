UI design "On Track"

On Track is a calendar tool for CUNY Students. Determined to simplify the time management of students and improve the efficiency of communication between students. This APP pursues a simple and efficient interface. Simplify the steps of setting up the calendar by synchronizing the CUNY blackboard. Because this project target for students, a simple and convenient communication page is necessary. Therefore, classrooms post a place for students communicating with each other. The system will automatically create a classroom based on the information synchronized from the blackboard, or user can create a classroom by themselves.
